# Config Files

This repository contains my personl configuration files for various tools and applications, managed with [GNU Stow](https://www.gnu.org/software/stow/).

- **[SketchyBar](https://github.com/FelixKratz/SketchyBar)** - Menu bar replacement with monitoring widgets
- **[Frame](https://github.com/tvanreenen/frame)** - Workspaces and tiling window manager
- **[Ghostty](https://github.com/mitchellh/ghostty)** - Faster, more configurable alternative to Terminal.app
- **[Zsh](https://www.zsh.org/)** - Shell config (aliases, functions, environment)
- **[Starship](https://starship.rs/)** - Shell prompt to show runtime and other contextual information
- **[Neovim](https://neovim.io/)** - Hyperextensible Vim-based text editor
- **[Homebrew](https://brew.sh)** - A broader set of packages and applications managed through a Brewfile
