# Config Files

This repository contains my personl configuration files for various tools and applications, managed with [GNU Stow](https://www.gnu.org/software/stow/).

- **[SketchyBar](https://github.com/FelixKratz/SketchyBar)** - Transparent menu bar replacement with system monitoring widgets
- **[Starship](https://starship.rs/)** - Cross-shell prompt
- **[AeroSpace](https://github.com/nikitabobko/AeroSpace)** - i3-like tiling window manager
- **[Ghostty](https://github.com/mitchellh/ghostty)** - Terminal emulator
- **[Neovim](https://neovim.io/)** - Hyperextensible Vim-based text editor
- **[Homebrew](https://brew.sh)** - Package manager
